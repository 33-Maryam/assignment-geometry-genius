1. What is DOM and what is the purpose of DOM?

Full form of DOM is the ''Document Object Model." Dom is use for interfacing web documents it helps user to communicate with a computer. Here document is HTML and object is the elements of HTML and the model is structure of web document where 'html' tag is the root tag and other other element into the 'html' tag is called child element or child node including nested child node. We can change the style of the structure through by calling the element that makes the webpage interactive and dynamic. Dom connected different page with parent page. Dom create structural representation of interactive document

2. How will you select HTML elements using DOM? Name the DOM methods.

We can manipulate html elements with JavaScript. For manipulating html elements first have to select the element. We can select the html element through several methods:

  • getElementsByTagName()

    • By using this method we will find the html tag and manipulate the element.The problem on which using this method is it select all tag in html which naming the selecting tag. Usually this method is use less than other. Example, const element = document.getElementsByTagName("p");

  • getElementsById()

    • This is the easiest method to select element on the DOM. It's work as CSS style where in html tag give a unique id by which CSS style work same this id is use in DOM for select the element. By which we will get specific tag for manipulating.This method is use frequently. Example: const element = document.getElementById("intro");

  • getElementsByClass()

    • This method is select more than one element which having same class name. If any elements need same type of manipulation that time this method is work. It will give you array like object. Example: const x = document.getElementsByClassName("detail-container");
    this 'detail-container' class will applicable for same named class and will return a list of element.

  • querySelectorAll

    • In querySelectorAll for select a CSS id need to call the id same as CSS style as it is manipulate whole CSS id list. This also called node list cause it is static and work for each of the value of the id as object. This method is work in for loop function. Example: const x = document.getElementsByClassName(".detail-container li");
    this 'detail-container li' class will manipulate the whole li from the class.

  • querySelectorl

    • querySelector use for select the first one from the class. Work as getElementById. We will manipulate individually by this method. Example: const x = document.getElementsByClassName(".detail-container li");
    this 'detail-container li' class will manipulate the first li from the class.

    4. What is a callback function and why will you use it?

    Callback function is use for pass an argument to another function. The call back function is start operating when another function finished operating. In javascript common callback function is event handle. It make the element more interactive. Callback function display more command by clicking a simple button so this make the web document dynamic and interactive.
